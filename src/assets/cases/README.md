# Картинки кейсов (src/assets/cases)

Фото учеников для блока «Истории учеников» (кейсы/отзывы).

Нужны 3 файла (PNG):

| Файл              | Кейс                                                  |
| ----------------- | ----------------------------------------------------- |
| `cases/1.png`     | Кейс с `id: 1` из `src/content/ru.json` → `cases`     |
| `cases/2.png`     | Кейс с `id: 2`                                         |
| `cases/3.png`     | Кейс с `id: 3`                                         |

## Как заменить фото

1. Подготовьте PNG (рекомендуется ширина 1080–1280 px).
2. Назовите файл `1.png`, `2.png` или `3.png` — по `id` кейса.
3. Положите его в эту папку (`src/assets/cases/`), заменив старый.

Имя файла для каждого кейса задаётся в `src/content/ru.json` → `cases[].image`
(например `"image": "cases/1.png"`).

## Если фото нет

Бот **не падает**: отправит текст кейса без картинки и запишет предупреждение
в лог (`[scheduler] media asset not found: cases/1.png; sending text only`).
Добавите файл — кейс начнёт приходить с фото.
